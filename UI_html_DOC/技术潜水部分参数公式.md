方案二：水面压力参考（展现升水风险预测，与16组织仓柱状图联动）如果你的 16 组织仓柱状图采用了上一阶段讨论的水面参考模型（Surface M-value Limit），那么强烈建议这两个大卡片也采用方案二。这样可以保证整个屏幕的条形图（Bar）拥有统一的物理参照物，不会让潜水员产生认知混淆。COMP_TISSUE_RAW_4012 (TISSUE 原始)$$\text{TISSUE\_RAW\%} = \frac{P_{\text{组织}} - P_{\text{水面}}}{M_0 - P_{\text{水面}}} \times 100\%$$物理意义：如果现在立刻升到水面，领航组织压力将达到水面原始 $M_0$ 值的百分之几。超过 100% 意味着不进行减压直接升水必然触发减压病风险。COMP_TISSUE_GF_4012 (TISSUE GF)$$\text{TISSUE\_GF\%} = \frac{P_{\text{组织}} - P_{\text{水面}}}{GF_{\text{high}} \times (M_0 - P_{\text{水面}})} \times 100\% = \frac{\text{TISSUE\_RAW\%}}{GF_{\text{high}}}$$物理意义：如果现在立刻升到水面，领航组织压力将达到你设定的水面最大保守度红线（GF High）的百分之几。当这个条形图触及 100% 时，免减压时间（NDL）刚好归零，系统开始提示减压停留。
